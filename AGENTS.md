# Boris / OpenClaw project rules for Codex

- Работай только внутри этого репозитория.
- Сначала определяй слой и blast radius.
- Сначала ищи source of truth.
- Не правь generated/runtime файлы, если есть master.
- Не делай широкий рефакторинг вместе с одной задачей.
