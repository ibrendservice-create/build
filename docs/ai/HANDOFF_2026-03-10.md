## 7. Что нельзя трогать

### КАТЕГОРИЧЕСКИ НЕЛЬЗЯ

1. **`/var/lib/apps-data/openclaw/data/.openclaw/openclaw.json`** -- не редактировать через `docker exec`. Использовать только внешний патч-скрипт + `docker compose restart openclaw`. K() перезапишет при старте контейнера.

2. **`/etc/nginx/sites-enabled/n8n-public-edge`** -- содержит `location /webhook/` proxy. Удаление -> все Okdesk webhooks 404. Инцидент 2026-03-04.

3. **WF3 (eHRHeiYttQgUgkHK)** -- 127 нод, файл-парсер. Изменять только через n8n UI с пониманием всей цепочки (LlamaParse -> Docling -> Vision -> evaluator).

4. **`fix-model-strategy.py`** -- восстанавливает конфиг после K(). Удаление -> потеря всех настроек при каждом рестарте контейнера.

5. **TG Bot webhook** -- должен указывать ТОЛЬКО на `https://n8n.brendservice24.ru/webhook/telegram-bot`. Пустой/неверный -> Boris не получает сообщения.

6. **DB schema `technicians`** -- 5809 записей, 30 колонок, 7 индексов, referenced by 4 таблицы. Менять ТОЛЬКО с миграцией. Завязаны: WF9, WF10, sync-скрипты.

7. **Cron `sync-pg-from-s2.sh`** и `sync-executors-from-s2.sh` -- sync S1<->S2. Удаление -> desync данных.

### ОСТОРОЖНО (менять с пониманием)

8. **WF10 Handle Dispatch Callback** -- 69507 chars JavaScript. 16+ action handlers. Изменения только с полным чтением и тестированием.
